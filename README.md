# DexScrapper

A Python-based web scraper project.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the scraper:
   ```bash
   python main.py
   ```

## Structure

- `src/scraper.py`: Core scraping logic.
- `src/utils.py`: Utility functions.
- `main.py`: Entry point.
